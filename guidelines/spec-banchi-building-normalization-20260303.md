# 番地・建物名自動整形仕様（2026-03-03）

## 対象
- `src/app/components/DataEntryForm.tsx`

## スコープ
- 基本モードの `banchi` 入力で、`1丁目7番1号` のような表記を `１ー７ー１` 形式に自動整形する。
- 住民票モードの `departBanchi` / `registryBanchi` でも同じ整形を行う。
- 基本モードの `building` 入力で、全角数字を半角数字へ自動変換する。
- 住民票モードの `departBuilding` / `registryBuilding` でも同じ変換を行う。
- 基本モードの `building` 入力で、半角英字を全角英字へ自動変換する。
- 住民票モードの `departBuilding` / `registryBuilding` でも同じ変換を行う。

## 制約
- 既存の補完表示、キー操作、保存処理を壊さない。
- サーバー処理は追加しない。
- 数字・英字変換以外の建物名文字は維持する。

## 受け入れ条件
- `banchi` に `1丁目7番1号` を入力すると、`１ー７ー１` になる。
- `banchi` に `１丁目７番１号` を入力しても、`１ー７ー１` になる。
- `building` に `１０１` を入力すると、`101` になる。
- `building` に `A棟` を入力すると、`Ａ棟` になる。
- 住民票モードの番地・建物名欄でも同等に変換される。

## 非対象
- 番地の補完候補ロジック変更。
- 既存CSVデータの書き換え。
