# 基本モード ファイル名固定化仕様（2026-03-05）

## 背景
- 基本モードで入力者名と同様に、ファイル名も固定入力したい。

## スコープ
- 設定画面に「ファイル名を固定する」トグルを追加する。
- 設定画面に「固定するファイル名」入力欄を追加する。
- 基本モードのファイル名入力欄は固定ON時に固定値表示＋編集不可にする。
- 基本モード保存/書き込み時は固定ONなら固定値を使用する。

## 制約
- 住民票モードの挙動は変更しない。
- 既存の入力者固定機能は維持する。

## 受け入れ条件
- 固定ON時、基本モードのファイル名入力欄がグレーアウトされる。
- 固定ON時、リスト保存・シート書き込みに固定ファイル名が使われる。
- 固定OFF時は従来どおり手入力できる。
