# 仕様書: 基本モード氏名の空白を全角化（2026-03-06）

## 背景
- 基本モードの氏名入力で、半角スペースが混在すると表記揺れが起きる。

## スコープ
- 基本モードの `氏名` フィールド（`name`）入力時に、半角スペースを必ず全角スペースへ変換する。
- 入力・貼り付けどちらでも `name` の onChange を通る値に適用する。

## 制約
- 対象は基本モードの `name` のみとし、他フィールドの空白挙動は変更しない。

## 受け入れ基準
- `山田 太郎` と入力すると `山田　太郎` になる。
- 連続半角スペースも連続全角スペースになる。

## 非スコープ
- 住民票モードの名前系フィールドの変換
- タブ・改行など空白全般の正規化
