# 仕様書: 基本モード シート2 Enter移動順変更（2026-03-06）

## 背景
- 基本モードのシート2入力では、実際の入力手順に合わせて Enter 移動順を調整したい。

## スコープ
- 基本モードかつシート2選択時のみ、Enter/↓/↑ のフォーカス移動順を以下に変更する。
- 期待順: 入力者 → ファイル名 → 肩書 → 氏名 → 会社名 → 郵便番号 → 都道府県 → 市区町村 → 町域 → 大字 → 字 → 小字 → 番地 → 建物名 → 電話番号 → 備考

## 制約
- シート1（従来基本モード）の移動順は変更しない。
- 既存の Shift+Enter で詳細項目を含む挙動は維持する。

## 受け入れ基準
- シート2モードで `ファイル名` から Enter を押すと次は `肩書` に移動する。
- シート2モードで `会社名` から Enter を押すと `郵便番号` に移動する。
- シート1モードでは従来の順序のまま。

## 非スコープ
- フィールド表示順そのものの変更
- 住民票モードの移動順変更
