# 基本モード シート2 入力欄列順配置仕様（2026-03-05）

## 背景
- 基本モードでシート2を選択したとき、入力欄の並びをシート列順に合わせたい。

## スコープ
- シート2選択時の入力欄順序を `A→B→C→D→E→F→G→H→I→J` に合わせる。
- 対象列:
  - A: 入力者
  - B: ファイル名
  - C: 肩書
  - D: 氏名
  - E: 会社名
  - F: 郵便番号
  - G: 住所1
  - H: 住所2
  - I: 電話番号
  - J: 備考

## 制約
- シート1（基本シート1）の既存入力レイアウトは変更しない。
- 書き込みマッピングのロジックは変更しない（UI配置のみ変更）。

## 受け入れ条件
- シート2選択時、肩書/氏名/会社名が郵便番号より前に表示される。
- シート2選択時、電話番号は住所関連の後に表示される。
- シート1選択時の見た目と挙動は従来どおり。
