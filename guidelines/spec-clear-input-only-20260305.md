# クリアボタンの入力欄限定クリア仕様（2026-03-05）

## 背景
- クリア操作時に、入力欄以外の状態まで消えると作業継続性が下がる。
- 画面上の入力フォーム（赤枠相当）のみを対象にしたい。

## スコープ
- 基本モード・住民票モードともに、`クリア` ボタンはフォーム入力値のみ初期化する。
- 固定設定が有効な項目は固定値を維持する。

## 非スコープ
- 保存済みリストの削除
- 編集中状態の解除
- シート同期メッセージの初期化
- PDFプレビュー状態のクリア

## 受け入れ条件
- `クリア` 後、フォーム項目だけが空になる（固定項目は固定値のまま）。
- PDFを読み込んでいる場合でも、`クリア` 後にPDFプレビューは保持される。
- 既存の書き込み・保存・コピー機能に影響しない。
