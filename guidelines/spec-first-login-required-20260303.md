# 初回利用時簡易ログイン仕様（2026-03-03）

## 対象
- `src/app/components/DataEntryForm.tsx`
- `package.json`
- `package-lock.json`

## スコープ
- アプリ利用前に簡易ログイン画面を追加する。
- 初回利用時のみログインを必須化し、認証成功後は同一ブラウザで再ログイン不要にする。
- 認証情報は以下で固定する。
  - name: `admin`
  - pass: `chihiro`
- 認証済み状態は `localStorage` に保存する。
- 認証失敗時はエラーメッセージを表示する。

## 制約
- 既存の入力機能、モード切替、PDF/シート/漢字表示を壊さない。
- サーバー追加や課金サービス利用は行わない。
- 実装はフロントエンド内で完結する簡易ログインとする。

## 受け入れ条件
- 初回アクセス時はログイン成功までツール本体を表示しない。
- `admin / chihiro` でログイン成功し、ツール本体が表示される。
- 間違った認証情報ではログインできず、エラーが表示される。
- 認証成功後に再読み込みしてもログイン画面は表示されない。
- `npm run build` が成功する。

## 非対象
- 強固な認証・暗号化・ユーザー管理。
- ログアウト機能。
- 複数ユーザー権限管理。
