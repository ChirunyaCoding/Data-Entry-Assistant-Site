# 仕様書: Google Map検索仕様の巻き戻し（2026-03-07）

## 背景
- 番地欠落対策として追加した地図検索クエリ補正を廃止し、以前のGoogle Map検索仕様へ戻す。

## スコープ
- 地図検索は Google Geocoding の整形住所をそのまま埋め込み検索に使用する。
- `検索クエリ` 表示を廃止し、表示は `検索住所` のみとする。
- 郵便番号付き検索入力の補助処理を削除する。
- 番地省略判定の注記表示を削除する。

## 制約
- 基本モード・住民票モードの両方に同じ巻き戻しを適用する。
- Geocoding API を使う現在のフロー自体は維持する。

## 受け入れ基準
- 地図表示時、埋め込み検索URLは Google 整形住所を使う。
- 地図下の表示は `検索住所: ...` の単一表示に戻る。

## 非スコープ
- 地図検索精度自体の再改善
- 別の候補選択UIの追加
