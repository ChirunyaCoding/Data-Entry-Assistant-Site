# Google Maps埋め込み表示（Maps Embed API）仕様（2026-03-06）

## 背景
- 住所入力後に外部タブ遷移ではなく、画面内で地図を確認したい。

## スコープ
- Google Maps表示方式を「新規タブ検索」から「Maps Embed APIのiframe埋め込み」に変更する。
- 基本モード・住民票モード（転出/本籍）それぞれで表示できるようにする。
- APIキーは `.env` の `VITE_GOOGLE_MAPS_EMBED_API_KEY` を参照する。

## 制約
- APIキー未設定時はエラーメッセージを表示し、埋め込みを実行しない。
- 既存の住所チェック機能には影響しない。

## 受け入れ条件
- 住所入力済みで地図表示操作すると、フォーム内にGoogle Maps iframeが表示される。
- APIキー未設定時に明確なエラーメッセージが出る。
- ビルドが成功する。
