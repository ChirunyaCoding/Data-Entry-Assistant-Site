# 保存済みリスト順序移動仕様（2026-03-04）

## 対象
- `src/app/components/DataEntryForm.tsx`
- `package.json`
- `package-lock.json`

## スコープ
- 基本モードの保存済みリストに `上へ` / `下へ` 移動操作を追加する。
- 住民票モード（住民票シート1）の保存済みリストに `上へ` / `下へ` 移動操作を追加する。
- 並び順は画面上のリスト順に反映し、既存の編集・削除・上書き書き込み機能と共存させる。

## 制約
- 住民票シート2の既存リスト（氏名入力欄）動作は変更しない。
- 既存のWebhook送信仕様は変更しない。

## 受け入れ条件
- リスト各項目で上へ/下へを押すと、その1件だけが隣接位置へ移動する。
- 先頭項目の上移動、末尾項目の下移動は無効化される。
- 移動中に書き込み系の同時実行で状態競合が起きない。

## 非対象
- 並び順の永続化（localStorage/サーバ保存）
- ドラッグアンドドロップ対応
