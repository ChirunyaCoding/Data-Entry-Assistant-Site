# Google検索結果住所の表示仕様（2026-03-06）

## 背景
- 地図下に入力住所ではなく、Google検索で実際にヒットした住所を表示したい。

## スコープ
- 地図表示時に Geocoding API を呼び、先頭結果の `formatted_address` を取得する。
- 基本モード・住民票モード（転出/本籍）で、取得した正式住所を地図下に表示する。
- 取得失敗時はエラーを表示し、地図/住所表示を更新しない。

## 制約
- APIキーは既存設定 `googleMapsEmbedApiKey` を利用する。
- Geocoding API が未有効なキーではエラーになるため、案内文を表示する。

## 受け入れ条件
- 地図表示後、地図下には Google の検索結果住所（formatted_address）が表示される。
- 入力変更やクリア時は表示住所をリセットする。
- ビルドが成功する。
