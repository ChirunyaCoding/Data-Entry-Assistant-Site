# Google Maps埋め込み下に検索住所表示 仕様（2026-03-06）

## 背景
- 地図表示後に、実際に検索した住所文字列を確認したい。

## スコープ
- 基本モード・住民票モード（転出/本籍）で、地図iframe読み込み後に「検索住所」を地図下へ表示する。
- 住所入力変更/クリア時は表示中地図と検索住所表示をリセットする。

## 制約
- クロスオリジン制約により、Google Maps iframe内の確定住所文字列を直接取得する実装は行わない。
- 表示する住所は検索に使用した住所文字列（入力連結値）とする。

## 受け入れ条件
- 地図表示ボタン押下後、地図が読み込まれたら地図下に検索住所が表示される。
- 入力変更やクリアで古い検索住所表示が残らない。
- ビルドが成功する。
