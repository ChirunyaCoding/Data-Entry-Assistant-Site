# リロード時リスト保持 + 書き込み後入力クリア 仕様（2026-03-05）

## 背景
- リロード保持を有効化しても、保存済みリストが復元されない。
- 書き込み実行後に入力欄を自動でクリアしたい。

## スコープ
- リロード保持機能に以下の状態を追加する。
  - 基本モード保存済みリスト
  - 住民票モード保存済みリスト
  - 住民票シート2の編集中リスト（フォルダ読み込み後の一覧）
- 基本モードの「書き込み」実行成功後に入力欄をクリアする。
- 住民票モード（シート1）の「書き込み」実行成功後に入力欄をクリアする。

## 制約
- リスト行の個別上書き書き込み（保存済みリスト側の操作）では従来通り入力欄を維持する。
- 既存のシート書き込みペイロード仕様は変更しない。

## 受け入れ条件
- リロード保持オン時、リロード後に保存済みリストが復元される。
- 基本/住民票シート1の「書き込み」成功後、入力欄がクリアされる（固定項目は維持）。
- ビルドが成功する。
