# リロード時の入力データ/PDF表示保持設定 仕様（2026-03-05）

## 背景
- ページをリロードすると、入力中データと表示中PDFが失われる。
- 設定でオンにした場合のみ、リロード後に復元したい。

## スコープ
- 設定に「リロード後も入力データとPDF表示を保持する」トグルを追加する。
- トグルがオンのとき、以下を保存・復元する。
  - 基本モード入力データ
  - 住民票モード入力データ
  - 現在モード（基本/住民票）
  - 右ペイン表示タブ（PDF/シート/漢字）
  - 表示中PDF
- トグルがオフのとき、保持データを削除し、以後の保持を行わない。

## 制約
- 既存の設定保存挙動は維持する。
- 「クリア」ボタンの仕様（入力のみクリア）は変更しない。
- PDFは容量制約回避のため IndexedDB へ保存する。

## 受け入れ条件
- トグルオンで、リロード後に入力値とPDF表示が復元される。
- トグルオフで、リロード後に入力値/PDF表示は復元されない。
- ビルドが成功する。
