# 住民票シート2 B列大文字化トグル仕様（2026-03-04）

## 対象
- `src/app/components/DataEntryForm.tsx`
- `package.json`
- `package-lock.json`

## スコープ
- 設定に「住民票シート2のB列ファイル名を大文字化する」オン/オフを追加する。
- 住民票シート2の書き込み時、B列値の生成で設定フラグを参照する。
- 設定は既存と同様に `localStorage` へ保存し、再読み込み後も維持する。

## 制約
- 住民票シート1および基本モードの書き込み仕様は変更しない。
- 既存のシートURL/Webhook設定項目は変更しない。

## 受け入れ条件
- トグルONで住民票シート2のB列が英字大文字として書き込まれる。
- トグルOFFで住民票シート2のB列が元の文字ケースで書き込まれる。
- 個別上書き書き込みと一括書き込みの両方で同じ挙動になる。
- `npm run build` が成功する。

## 非対象
- 既存データの一括再変換
- Apps Script側ロジック変更
