# 住民票シート2 UI切替とファイル書き込み仕様（2026-03-04）

## 対象
- `src/app/components/DataEntryForm.tsx`
- `guidelines/resident-sheet-webhook-setup-20260303.md`
- `guidelines/spec-resident-sheet2-name-only-and-folder-write-20260304.md`
- `package.json`
- `package-lock.json`

## スコープ
- 住民票モードで「住民票シート2」を選択した場合、左入力欄を氏名入力のみのUIに切り替える。
- 住民票シート2では、フォルダ読み込み時にファイル数分の氏名入力欄を生成する。
- 住民票シート2の書き込みは以下で行う。
  - `B3` から下へ: ファイル名（拡張子付き）
  - `C3` から下へ: 氏名入力欄の値
- ファイル名を書き込む際、文字は強制的に全角化する（漢字/ひらがな/カタカナ/英字/記号/数字）。

## 制約
- 住民票シート1の既存仕様（B/F/G/H/I/J/K/L、6行目開始）は維持する。
- 住民票シート2の通常フォーム入力項目は表示しない（氏名入力UIのみ）。
- サーバー常駐は追加しない。既存Apps Script Webhookを拡張する。

## 受け入れ条件
- 住民票シート2選択時に、左入力欄が氏名入力UIへ切り替わる。
- フォルダ読込後、`tif/tiff`件数と同数の氏名入力欄が生成される。
- 住民票シート2の書き込みで、`B3:C`へ件数分追記される。
- ファイル名は全角化されて保存される。
- `npm run build` が成功する。

## 非対象
- 基本モードの既存書き込み仕様変更
- 住民票シート1の項目構成変更
