# 住民票シート反映 成功判定厳密化仕様（2026-03-03）

## 対象
- `src/app/components/DataEntryForm.tsx`
- `package.json`
- `package-lock.json`

## スコープ
- 住民票モードのシート反映レスポンス判定を厳密化する。
- `ok: true` のJSONレスポンス以外は失敗として扱う。
- 成功時は反映行番号を画面に表示する。
- 非JSONレスポンス（例: ログインHTML）時は設定不備として明示エラーを出す。

## 制約
- 既存の列マッピングと送信ペイロード構造は維持する。
- 既存の保存済みリスト追加機能は維持する。

## 受け入れ条件
- シート未反映時に成功メッセージが出ない。
- レスポンスがHTML等のとき、ユーザーに設定見直しエラーを表示する。
- 反映成功時に「何行目へ反映したか」が表示される。
- `npm run build` が成功する。

## 非対象
- Apps Script実装自体のロジック変更。
