# シート書き込みフォントサイズ設定仕様（2026-03-05）

## 背景
- 基本モード/住民票モードのシート書き込み時フォントサイズを固定値ではなく設定で変更したい。

## スコープ
- 設定画面に「シート書き込みフォントサイズ」を追加する。
- 基本モード・住民票モードの全シート書き込みペイロードへ `fontSize` を含める。
- Apps Script側で `fontSize` を受け取り、未指定時は既定値 `10` を使用する。

## 制約
- 既存の書き込み列マッピングは変更しない。
- フォントファミリー（Meiryo）は維持する。
- 設定値は整数として扱い、範囲外は安全な範囲に丸める。

## 受け入れ条件
- 設定値を変更すると、基本モード/住民票モード双方の書き込みフォントサイズが反映される。
- 設定未変更時は従来どおり `10` で書き込まれる。
